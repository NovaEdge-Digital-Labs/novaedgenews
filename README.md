# NovaEdge News

NovaEdge News is a modern news platform featuring an AI-driven editorial pipeline, daily newsletters, and automated SEO optimization.

## Project Structure

- **`frontend/`**: Next.js user-facing platform.
- **`backend/`**: NestJS server handling data, AI writing, and automation schedules.

## Getting Started

Please refer to [**run.md**](./run.md) for detailed instructions on local setup, environment configuration, and deployment.

## Key Features

- **AI Writing Engine V2**: Natural tone and high-quality news analysis.
- **Automated Newsletter**: Scheduled daily dispatch via SendGrid.
- **SEO Automation**: Dynamic metadata, JSON-LD schema, and sitemap generation.
- **Real-time Monitoring**: System alerts via Telegram and Email.
